# Musiques d'ambiance

Placez ici vos fichiers audio de musique de fond pour vos séances de cohérence cardiaque.

## Formats supportés
- MP3
- WAV
- OGG
- M4A

## Types de musiques recommandées
- Musique méditative
- Sons binauraux
- Musique ambient / drone
- Fréquences de guérison (432Hz, 528Hz, etc.)
- Sons de la nature (forêt, océan, pluie)
- Musique classique apaisante
- Musique new-age

## Conseils
- Privilégiez des morceaux sans changements brusques de tempo ou de volume
- Les pistes de 10-30 minutes sont idéales
- Un volume modéré (30-40%) est recommandé pour ne pas couvrir les sons de respiration

## Note
Les fichiers placés dans ce dossier pourront être chargés automatiquement dans une future version de l'application.

Pour l'instant, utilisez le bouton "Ajouter des musiques de fond" dans l'interface pour uploader vos fichiers.
